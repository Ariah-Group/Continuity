• This screen is available only to System Administrators and Local Administrators.
  • Role acronyms-
    • SA - System Administrator
    • LA - Local Adminstrator
    • UV - Universal Viewer
    • LU - Local User (default role)
  • A System Administrator
    • Can enable/disable each location's access to UC Ready
    • Can add/remove other System Administrators
    • Can add/remove Local Administrators for any location
    • Has query access to all data in the UC Ready database (all locations)
    • Is automatically a Local Administrator for his/her home location.
  • A Local Administrator
    • Has access to the Administrator's Control Screens for his/her location
    • Can enable/disable his/her location's access to UC Ready
    • Has read/write access to all plans created at his/her location
    • Has exclusive ability to delete entire plans at his/her location
    • Can add/remove other local administrators and universal viewers for his/her location
    • Can download documents for his/her location on the Download Documents screen
    • Has query access to all data in the UC Ready database for his/her location only.
  • Universal Viewers
    • The Universal Viewer role exists primarily for times of crisis. The idea is to have a number of staff pre-authorized to access documents if a Local Administrator is not available.
    • All Local Administrators will automatically appear on the list of Universal Viewers.
    • Universal Viewers access documents via the Choose a Plan screen.
    • Because some Universal Viewers may access the UC Ready tool only rarely, it may be advisable to keep printed instructions handy for them.
