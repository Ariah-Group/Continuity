Setup Acronyms

Your list of plans will eventually get quite long. Acronyms can be used to group the names of plans (recall that the name of the plan is simply the name of the department it applies to).

For example, suppose SOM is the common acronym for your School of Medicine. The plans of the many departments in that School can be made to sort together on your list of plans by naming them as follows

  • SOM - Dermatology
  • SOM - Internal Medicine
  • SOM - Orthopedic Surgery

To achieve this, you should specify the desired acronyms below. When a user creates a plan, he/she will be asked to select from your list of acronyms. If the user selects an acronym, the plan name will thereafter appear in the form above (e.g. "SOM - Dermatology").

If the user does NOT select an acronym, the plan name will appear without one ("Dermatology").

To modify an existing item, first highlight that row on the grid then use the entry-forms at bottom to modify.
