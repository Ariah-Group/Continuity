• The question "Identify Your Unit's Critical Functions" is the most crucial question in this planning tool. This screen enables you (as Local Administrator) to select good examples of critical function lists from existing plans that will then be displayed to users to aid their thought process.
  • It may take time to accumulate some good examples; until you reach that point, you should uncheck the option called Show Examples of Critical Functions on the System Settings page. Doing so will hide this feature from users until you have provided the examples.
